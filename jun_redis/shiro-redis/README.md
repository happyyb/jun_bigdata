**shiro-redis**

shiro集成redis的适配器，为解决shrio-ehcache不利于集群而打造的缓存集群方案。

**Maven坐标**
```
<dependency>
    <groupId>org.iherus.shiro</groupId>
    <artifactId>shiro-redis</artifactId>
    <version>1.0.0</version>
</dependency>
```
